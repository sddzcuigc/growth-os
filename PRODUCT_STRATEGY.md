# 成长OS：百万美元产品计划

## 1. 产品定位

成长OS不是待办清单，也不是AI家教。它是一个面向6-12岁孩子的“个人成长操作系统”：每天通过短对话理解状态，推荐一个恰当行动，把完成、困难、兴趣和复盘沉淀为可修正的长期成长证据。

核心承诺：孩子越来越懂自己，家长越来越少靠催促，系统越来越会给出刚刚好的下一步。

## 2. 目标用户与付费者

- 核心用户：6-12岁、有手机或平板使用条件的孩子。
- 核心付费者：关注自主学习、执行功能、AI素养和项目能力的家长。
- 第二付费者：小型教育机构、成长教练、家庭教育顾问。
- 首个滩头市场：一二线城市、愿意为儿童成长订阅付费、已有AI工具使用习惯的家庭。

## 3. 真正的产品闭环

1. AI先问：理解孩子今天的能量、兴趣、困难和时间。
2. 只给一步：推荐一个可完成、可观察、有产出的任务。
3. 行动留证据：完成、跳过、卡住、喜欢和作品都成为行为证据。
4. 记忆进化：系统区分一次性状态、重复趋势和稳定策略，并保留来源与置信度。
5. 调整路线：技能树、周计划和任务难度随证据更新。
6. 解释变化：孩子看到“我发现了什么”，家长看到“最近发生了什么”，双方都能纠正AI。

灵感也属于闭环，但不被强行任务化。发现页一次只唤醒一颗旧灵感；孩子可以试10分钟、继续收藏、30天后再看或明确放下。每次选择写入数据库并改变候选集，避免AI反复催促。

## 4. 百万美元价值假设

以年订阅699元计算，约1.05万付费家庭对应约100万美元年经常性收入。更现实的阶段目标：

- 0-3个月：100个种子家庭，验证孩子7日留存和任务完成率。
- 4-6个月：1,000个活跃家庭，验证月付转年付和家长周报价值。
- 7-12个月：5,000个付费家庭，建立转介绍与内容合作渠道。
- 12-24个月：10,000-15,000个付费家庭，形成家庭版与机构版双产品线。

建议定价：免费版每天1次推荐；家庭版49元/月或399-699元/年；专业版按教练席位和孩子数量收费。

## 5. 北极星指标

北极星指标：每周完成“提问 → 行动 → 复盘”闭环的孩子数。

关键指标：

- 激活：注册24小时内完成首次个性化任务。
- 价值：7天内完成3次闭环，且至少一次主动点击“我想继续”。
- 留存：D7孩子留存≥35%，家庭周报打开率≥60%。
- 质量：推荐接受率≥55%，连续三次同质任务率≤10%。
- 安全：错误标签纠正入口100%可见；高风险内容进入推荐率为0。
- 商业：免费转付费≥5%，年订阅续费≥60%，自然转介绍≥25%。

## 6. 产品路线图

### Phase A：可信MVP（现在至6周）

- 注册、角色、数据库、导出、长期记忆。
- 成长洞察：展示证据、趋势、可纠正记忆和本周变化。
- 家长同意与儿童隐私说明。
- 完整埋点：推荐、接受、完成、跳过、复盘、回访。
- 20个家庭连续使用测试，每周访谈5组。

退出标准：D7≥25%，每周闭环≥2，推荐接受率≥45%。

### Phase B：记忆进化（第7-12周）

- 记忆分层：事件、偏好、有效策略、成长假设。
- 每条假设保留证据来源、反证、置信度和更新时间。
- “我的使用说明书”已完成第一版：至少双证据形成场景化策略，孩子反馈决定是否进入AI上下文，原始证据删除会自动撤回。
- AI每周合并可共享证据，孩子可选择“像我 / 有点不像 / 下周试这个”；第一版成长罗盘已完成。
- 技能图根据证据更新，不根据完成次数机械升级。
- 生成孩子版“本周发现”和家长版“少催促建议”。

退出标准：洞察纠正参与率≥30%，周报带来的次周回访提升≥15%。

### Phase C：作品与真实世界（第4-6个月）

- 项目路线、作品上传、阶段里程碑和展示页。
- 任务可连接日历、附近活动、图书和安全新闻。
- 家庭协作任务有明确角色，不让家长变成监督员。
- 建立经过审核的任务模板市场。

### Phase D：商业化与平台（第7-12个月）

- 家庭订阅、试用、支付、权益和续费。
- 家长空间与孩子空间分离。
- 教练/机构控制台、授权角色、群组和审计日志。
- A/B实验、成本控制、模型路由和质量评估集。
- iOS/Android PWA封装、通知和离线任务。

## 7. 第一轮迭代清单

P0：注册登录、账户恢复、成长洞察、记忆可见可纠正与策略压缩、带引用和隐私边界的成长档案问答、数据导出、家长同意、事件埋点、第一人称成长日记（自主写、AI提问、基于草稿追问）、带证据与反证的成长假设、灵感到行动状态机、自适应行动台、保留决定历史且能校准当前节奏的非惩罚式任务协商、AI统一随手记（行动/灵感/感悟三向分流）、最多三个且由真实证据推进的成长方向、非惩罚式微习惯、可恢复专注舱、可纠正的每周成长罗盘、任务级体验反馈、难度校准与动力线索、带隐私撤回的作品证据库、只给一个下一步且支持语自适应的每日指挥台均已完成第一版。

P1：孩子授权的家长版少催促周报已完成第一版；下一步是作品版本对比与展示页、任务质量离线评估、模型成本监控、推荐评估集。

P2：支付、邀请、家庭协作、机构版、模板市场。

## 8. 不做什么

- 不做心理诊断、智力判断或天赋定论。
- 不用排行榜刺激孩子竞争。
- 不把完成数量当作成长本身。
- 不用新闻制造焦虑或追热点。
- 不在未经明确同意时共享、训练或出售儿童数据。

## 9. 每两周迭代方法

每个Sprint只追一个行为结果：先定义指标和失败条件，再上线最小功能；对5个家庭观察真实使用；根据行为数据和访谈决定保留、修改或删除。每次发布必须同时更新安全评估、推荐评估集和成本看板。
